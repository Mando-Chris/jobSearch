# JobSearch

A job search/skill search application created with React and Redux.

### Getting Started


Install the dependencies and start the server.

```
npm install
npm start
open http://localhost:3000
```

### Dependencies

* React: ^15.6.1
* React-redux: ^5.0.6
* Redux: ^3.7.2
* Redux-thunk: ^2.2.0
